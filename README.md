# Peabux School System

## Installation

```bash
$ git clone https://github.com/genbliz/peabux-school.git
$ cd peabux-school
```

## The Server

### Running the Server

```bash
$ cd school-server
$ npm install
$ npm run server
```
The server will be running on port http://localhost:3500 by default

## The App

### Running the app

```bash
$ cd school-app
$ npm install
$ npm run dev
```

The app will be running on port http://localhost:3000 by default
